# 🤖 Missão IA - Alura Start

Projeto interativo desenvolvido como parte da **Unidade de Funções** da Alura.  
O objetivo é explorar conceitos fundamentais de JavaScript como:

- ✅ Lista de objetos
- ✅ Funções para dividir problemas grandes em menores
- ✅ Manipulação do DOM com métodos JavaScript
- ✅ Reflexão sobre o uso da Inteligência Artificial no dia a dia

---

## 🧩 Funcionalidades

- **Lista de ferramentas IA** – exibe objetos com nome e categoria
- **Adicionar / Restaurar** – modifica a lista dinamicamente
- **Gerar reflexão** – mostra pensamentos aleatórios sobre IA
- **Manipular DOM** – aplica efeitos visuais nos elementos
- **Duplo clique** – reflexão personalizada sobre uma ferramenta

---

## 🚀 Como executar

1. Baixe todos os arquivos do projeto:
   - `index.html`
   - `style.css`
   - `script.js`

2. Abra o arquivo `index.html` em qualquer navegador moderno.

3. Interaja com os botões e explore o comportamento da aplicação.

---

## 🛠️ Tecnologias utilizadas

- HTML5
- CSS3 (com design inspirado na Alura)
- JavaScript (Vanilla)

---

## 📚 Conceitos aplicados

- **Objetos JavaScript** – lista de ferramentas com `id`, `name` e `category`
- **Funções** – cada ação (renderizar, adicionar, resetar, refletir) é uma função separada
- **Manipulação do DOM** – `getElementById`, `querySelector`, `createElement`, `appendChild`, `style`, `textContent`
- **Eventos** – `click`, `dblclick`, `setTimeout`

---

## 📝 Reflexão sobre IA

O projeto incentiva a reflexão sobre como a Inteligência Artificial está presente em tarefas cotidianas, desde a escrita e programação até a criatividade e organização pessoal.
